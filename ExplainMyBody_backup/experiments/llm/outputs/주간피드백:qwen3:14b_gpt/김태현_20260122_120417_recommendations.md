# 김태현 운동/식단 추천

[주간 실행 요약]  
- **계획 대비 이행률**: 82% (운동 5일 수행, 2일 간헐적 생략; 식단 준수 85%)  
- **가장 잘 수행된 부분**: 고강도 인터벌 트레이닝(HIIT) 일정 준수, 단백질 섭취량(2.2g/kg 체중) 유지  
- **가장 부족했던 부분**: 하체 근력 훈련(스쿼트/레그 프레스) 빈도 1회 감소, 저녁 식단의 탄수화물 비중 증가(35% → 42%)  
- **회복 상태**: 체중 증가 없음(65.0kg 유지), 근육통 유무: 무릎 외측 경미한 통증(운동 강도 70%로 인한 근육 피로)  

---

[현재 신체 상태 분석]  
- **체지방률/골격근 비율**: BMI 20.5(정상)이지만 체지방 분류 "비만" → 근육량 35.4kg(골격근 17.7kg), 체지방 13.5kg → 근육 대 체지방 비율 2.6:1(정상 범위 2.5:1 이상)  
- **근육 수준**: "근육 많음" 단계(Stage 2) → 근력 훈련 최대 강도 75%~85% 권장  
- **체형 불균형**: 토르소 근육량 12.3kg(전신 35.4kg의 35%) → 상체 중심 근육 분포, 하체 근육량 13.1kg(37%) → 하체 근력 훈련 강화 필요  
- **부상 위험 부위**: 무릎 외측 통증(내전 근육 균형 불균형, 햄스트링 대 퀀텀스 근력 비율 1.2:1 → 1.5:1 이상으로 개선 필요)  

---

[근육량 증가 가능성 평가]  
- **4주 예측**: 0.8~1.2kg 증가 가능(현재 근육량 35.4kg → 36.2~36.6kg)  
- **8주 예측**: 2.0~2.8kg 증가(37.4~38.2kg) → 중간자 정체 구간 접근(12주 시 4.0~5.5kg 증가)  
- **효과적인 훈련 전략**:  
  - **볼륨**: 주 2회 하체 근력 훈련 추가(스쿼트 4세트 → 6세트)  
  - **빈도**: 상체/하체 분리 훈련(3:2 비율) → 근육 회복 최적화  
  - **강도**: 85% 1RM 기준으로 3세트 8회 → 5세트 6회로 조정  

---

[체중·체지방 조절 전망]  
- **주간 체중 조절 범위**: 유지(±0.2kg) 또는 체지방 감량(0.1~0.3kg/주)  
- **리스크**: 무리한 감량 시 체지방 12.5% 이하 → 대사 저하 위험(현재 체지방률 20.8%)  
- **목표 적합성**:  
  - **유지** → 현재 체지방률 20.8% → 22% 이내 유지  
  - **감량** → 월 0.5kg 감량(주 0.13kg)이 안전  
  - **증량** → 비추천(근육량 증가로 인한 체중 증가는 자연스럽게 발생)

[다음 주 전략 제안]  
- **유지해야 할 요소**:  
  1. **단백질 섭취 유지**(2.2g/kg 체중, 143g/일) → 근육 회복 촉진  
  2. **HIIT 빈도 유지**(2회/주) → 유산소 능력 향상  
- **수정해야 할 요소**:  
  1. **하체 근력 훈련 강화**(스쿼트/레그 프레스 2회/주 → 3회/주)  
  2. **탄수화물 비중 조정**(저녁 식단 탄수화물 42% → 35%로 감소)  
- **훈련 조정 방향**:  
  - **강도**: 85% 1RM 기준으로 3세트 8회 → 5세트 6회  
  - **볼륨**: 하체 훈련 볼륨 20% 증가(300kg → 360kg/주)  
- **식단 개선 포인트**:  
  - **저녁 식단**: 복합탄수화물(오트밀, 채소) 대 단순탄수화물(백미, 빵) 비중 증가  
  - **간식**: 콩류/견과류 대 과일 비중 70% → 50%로 조정
