# 최강훈 운동/식단 추천

### [체형 분석 요약]  
**현재 상태 및 개선 방향**  
25세 남성, BMI 26.2(비만1단계)에 해당하며, 체지방률은 표준이지만 **골격근 비율 0.494**로 근육량이 충분한 상태입니다. Stage1(비만형)에서 Stage2(고근육체형)까지의 체형 분류를 보았을 때, **상체(복부)에 지방 축적이 주요 문제**로 보입니다. 근육량은 이미 충분하지만, 체지방률을 유지하면서 복부 지방을 줄이는 것이 목표입니다.  

**개선 방향**:  
- **상체 중심의 유산소 운동**으로 복부 지방 감소  
- **근력 운동으로 근육량 유지** (Stage2 체형 유지)  
- **고단백, 저탄수화물 식단**으로 체지방 증가 방지  

---

### [이번 주 운동 목표]  
1. **복부 지방 3% 감소** (체지방률 유지, 복근 윤곽 개선)  
2. **근력 유지** (골격근 비율 0.494 유지)  
3. **주 4회 유산소 운동** (20분 이상)  
4. **전신 근력 운동 3회** (복부, 허리, 다리 강화)  

---

### [주간 운동 스케줄]  
**월요일 (상체 근력 + 유산소)**  
- **체조 브이(V) 운동**: 3세트 × 15회 (가슴, 어깨, 상체 근력 강화)  
- **플랭크**: 3세트 × 30초 (복부 긴장 유지)  
- **점프잭**: 3세트 × 1분 (심폐 기능 향상)  

**화요일 (유산소 + 복부)**  
- **스터드(Steady State) 걷기**: 30분 (속도 5km/h)  
- **레그 레이즈**: 3세트 × 15회 (복부 지방 감소)  
- **다이아몬드 푸시업**: 3세트 × 10회 (삼두근, 가슴 강화)  

**수요일 (휴식)**  
- **스트레칭**: 30분 (전신 유연성 향상)  
- **수분 섭취**: 2L 이상 (체지방 감소 지원)  

**목요일 (하체 근력 + 유산소)**  
- **스쿼트**: 3세트 × 15회 (하체 근력, 복부 긴장)  
- **스탠딩 레그 레이즈**: 3세트 × 20회 (복부 지방 감소)  
- **스프린트**: 3세트 × 30초 (심폐 기능 향상)  

**금요일 (상체 근력 + 유산소)**  
- **체조 브이(V) 운동**: 3세트 × 15회 (가슴, 어깨 강화)  
- **플랭크**: 3세트 × 30초 (복부 긴장)  
- **점프잭**: 3세트 × 1분 (심폐 기능 향상)  

**토요일 (유산소 + 복부)**  
- **스터드 걷기**: 30분 (속도 5km/h)  
- **레그 레이즈**: 3세트 × 15회 (복부 지방 감소)  
- **다이아몬드 푸시업**: 3세트 × 10회 (삼두근, 가슴 강화)  

**일요일 (휴식)**  
- **수분 섭취**: 2L 이상  
- **유연성 운동**: 30분 (전신 스트레칭)

### [이번 주 식단 계획]  
**아침 (3가지 옵션)**  
1. **단백질 오믈렛 + 통밀 빵 + 케라스**  
2. **단백질 바 + 요구르트 + 아보카도**  
3. **두부 두부 볶음 + 채소 수프**  

**점심 (3가지 옵션)**  
1. **계란 후라이 + 채소 샐러드 + 퀴노아**  
2. **삼겹살 구이 + 김치 + 브로콜리**  
3. **연어 스테이크 + 고구마 + 시금치**  

**저녁 (3가지 옵션)**  
1. **닭가슴살 구이 + 채소 볶음 + 현미**  
2. **두부 수프 + 채소 스시**  
3. **연어 샐러드 + 고구마**  

**간식 (3가지 옵션)**  
1. **그릭요거트 + 견과류**  
2. **프로틴바 + 케라스**  
3. **두부 수프 + 채소**

### [실천 가이드]  
**식사 타이밍**:  
- 아침: 7~9시  
- 점심: 12~14시  
- 저녁: 18~20시  
- 간식: 10~11시, 15~16시  

**장보기 리스트**:  
- **고단백 식품**: 닭가슴살, 연어, 두부, 계란, 요구르트  
- **저탄수화물**: 현미, 퀴노아, 고구마  
- **채소**: 시금치, 브로콜리, 당근, 파프리카  

**Meal Prep 팁**:  
- **주말에 3일치 식사 준비**: 닭가슴살, 연어, 두부를 한 번에 조리 후 냉동 보관  
- **현미밥 한끼씩 냉동**: 1인분씩 분할 보관  
- **채소는 미리 세척**: 용기에 보관해 필요 시 사용  

**외식 시 팁**:  
- **한식**: 쌈밥, 생선구이 정식 (튀김 제외)  
- **양식**: 그릴드 치킨 샐러드, 연어 스테이크  
- **피할 것**: 튀김, 크림 소스, 단 음료
