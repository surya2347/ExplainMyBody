# 한지우 운동/식단 추천

### [체형 분석 요약]  
당신은 BMI 19.5(정상)이지만 **체지방 비만형**으로 분류되어 있어요. 이는 **다른 사람보다 체지방 비율이 높은 상태**를 의미합니다. 특히 **상체(몸통) 근육 수준은 표준 이상**이지만, **전체 근육량은 낮은 수준**(골격근 비율 43.6%)으로 보여요. 이는 **근육량 부족 + 체지방 과다**라는 **마른비만형** 체질로, **에너지 대사가 느리고 근력이 약한 상태**일 수 있어요.  

**개선 방향**:  
- **근육량 증가**로 대사 활성화  
- **상체 근력 강화**로 체지방 감소 촉진  
- **유산소 운동**으로 체지방 감소  
- **균형 잡힌 식단**으로 과도한 탄수화물/당 섭취 방지  

---

### [이번 주 운동 목표]  
- **근육량 증가**: 주 3회 근력 운동으로 근육량 1.5% 증가 (체지방 대사 촉진)  
- **체지방 감소**: 주 3회 유산소 운동으로 하루 30분 이상 활동  
- **상체 근력 강화**: 스쿼트, 플랭크, 팔굽혀펴기 등으로 상체 근력 10% 향상  
- **운동 습관화**: 주 3회 이상 운동 완료 (운동 기록 체크리스트 활용)  

---

### [주간 운동 스케줄]  
**월요일 (상체 근력 + 유산소)**  
- **운동**:  
  1. **플랭크 3세트** (1분/세트, 무릎 대지 않고 팔꿈치로 지지)  
  2. **팔굽혀펴기 3세트** (10회/세트, 무릎을 구부려도 OK)  
  3. **상체 레이즈 3세트** (15회/세트, 팔을 머리 위로 높이 들어 올림)  
  4. **유산소**: 걷기 30분 (가능하면 경사로 활용)  
- **Tip**: 플랭크 시 복부에 힘을 주고 허리가 떨어지지 않게 유지  

**화요일 (하체 근력 + 스트레칭)**  
- **운동**:  
  1. **스쿼트 3세트** (15회/세트, 무릎을 발끝 방향으로 향하게)  
  2. **벽 걷기 3세트** (10회/세트, 다리 뻗고 뒤로 걷기)  
  3. **다리 레이즈 3세트** (15회/세트, 복부에 힘 주고 무릎을 향상)  
  4. **스트레칭**: 햄스트링, 종아리, 허리 스트레칭 5분  
- **Tip**: 스쿼트 시 무릎이 발끝을 넘지 않도록 주의  

**수요일 (유산소 + 심폐 강화)**  
- **운동**:  
  1. **유산소**: 빠르게 걷기 또는 점프하는 걷기 30분 (심박수 120~140 유지)  
  2. **심폐 강화**: 계단 오르기 10분 (20~30계단 반복)  
- **Tip**: 유산소 시 숨을 크게 내쉬며 리듬 유지  

**목요일 (휴식 또는 가벼운 활동)**  
- **운동**:  
  1. **가벼운 활동**: 산책 20분 또는 요가 20분  
  2. **자세 교정**: 컴퓨터 앞에서 자세 교정 훈련 10분 (등을 곧게 펴고 허리를 펴기)  

**금요일 (상체 + 유산소)**  
- **운동**:  
  1. **플랭크 3세트** (1분/세트)  
  2. **팔굽혀펴기 3세트** (10회/세트)  
  3. **유산소**: 걷기 30분  
- **Tip**: 플랭크 시 복부에 무게 중심을 둔 채 유지  

**토요일 (하체 + 유산소)**  
- **운동**:  
  1. **스쿼트 3세트** (15회/세트)  
  2. **벽 걷기 3세트** (10회/세트)  
  3. **유산소**: 빠르게 걷기 30분  
- **Tip**: 스쿼트 시 허리가 굽지 않도록 주의  

**일요일 (휴식 또는 복근 강화)**  
- **운동**:  
  1. **복근 강화**: 복식 호흡 10회, 바이시클 크런치 15회  
  2. **자세 교정**: 컴퓨터 앞에서 자세 교정 훈련 10분

### [이번 주 식단 계획]  
**아침 (3가지 옵션)**  
1. **한식**: 계란 1개 + 김밥 1개 + 우유 1잔  
2. **양식**: 오트밀 50g + 생딸기 1개 + 블루베리 10개  
3. **일식**: 시금치 오믈렛 1개 + 현미밥 1공기 + 콩수프 1그릇  

**점심 (3가지 옵션)**  
1. **한식**: 갈비탕 1그릇 + 김치 1/4공기 + 두부 1/2개  
2. **양식**: 닭가슴살 샐러드 150g + 그릭요거트 100ml  
3. **일식**: 연어 회덮밥 1공기 + 시금치 1/2그릇  

**저녁 (3가지 옵션)**  
1. **한식**: 미역국 1그릇 + 찌개 1/2공기 + 김 1장  
2. **양식**: 치킨 그릴 150g + 퀴노아 50g + 케일 1/2그릇  
3. **일식**: 연어 초밥 3개 + 시금치 1/2그릇  

**간식 (3가지 옵션)**  
1. **한식**: 두부 1/2개 + 콩가루 요구르트 100ml  
2. **양식**: 블루베리 10개 + 아몬드 10알  
3. **일식**: 에도마키 1개 + 두부 요거트 100ml
