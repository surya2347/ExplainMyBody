# 박서연 운동/식단 추천

### [체형 분석 요약]  
40세 여성으로 BMI 28.0(비만1단계)에 해당하며, 체지방 비율이 높고 근육량이 부족한 상태입니다. 특히 **상체 근육이 표준 이상**으로 분포되어 있어, **하체와 상체 근력 균형**을 맞추는 것이 중요합니다. 체지방 분포는 전신에 걸쳐 균일하게 나타나고 있어, **전신 지방 감소**가 주요 목표입니다.  

### [이번 주 운동 목표]  
- **체중 감소**: 1주간 0.5kg 감소 목표 (주간 3~4회 유산소 운동 포함)  
- **근육량 증가**: 근력 훈련으로 **하체 근력 강화** 및 **상체 근육 균형 조정**  
- **유연성 향상**: 스트레칭 및 동적 워밍업으로 **근육통 최소화**  

---

### [주간 운동 스케줄]  

#### **월요일: 상체 근력 훈련 + 유산소**  
- **워밍업**: 5분 동적 스트레칭 (팔 흔들기, 어깨 회전)  
- **운동**:  
  1. **덤벨 푸시업** (2세트, 10회) → 상체 근력 강화  
  2. **벽 압박** (2세트, 30초) → 가슴 근육 활성화  
  3. **플랭크** (2세트, 20초) → 코어 안정성 향상  
- **유산소**: 걷기 30분 (속도는 4km/h)  

#### **화요일: 하체 근력 훈련 + 스트레칭**  
- **워밍업**: 5분 다리 스트레칭 (다리 뒤로 굽기, 햄스트링 스트레칭)  
- **운동**:  
  1. **체중 스쿼트** (3세트, 15회) → 하체 근력 향상  
  2. **스탠딩 레그 레이즈** (2세트, 15회) → 햄스트링 강화  
  3. **스플릿 스탠드** (2세트, 30초) → 균형 개선  
- **스트레칭**: 10분 정적 스트레칭 (무릎 당기기, 햄스트링 당기기)  

#### **수요일: 유산소 + 코어 운동**  
- **워밍업**: 5분 걷기 (3km/h)  
- **운동**:  
  1. **유산소**: 수영 또는 빠른 걷기 30분 (심박수 120~140 유지)  
  2. **코어**:  
     - **다이아몬드 플랭크** (2세트, 15초)  
     - **크런치** (3세트, 20회)  

#### **목요일: 유연성 향상 + 경량 근력**  
- **워밍업**: 5분 요가 동작 (산책자세, 고양이-사자 자세)  
- **운동**:  
  1. **체중 레그 레이즈** (2세트, 12회) → 하체 유연성 향상  
  2. **덤벨 레그 프레스** (2세트, 10회) → 하체 근력 강화  
- **스트레칭**: 10분 정적 스트레칭 (엉덩이, 종아리)  

#### **금요일: 전신 근력 + 유산소**  
- **워밍업**: 5분 동적 스트레칭 (팔 흔들기, 다리 스트레칭)  
- **운동**:  
  1. **체중 브이 자세** (3세트, 15회) → 전신 근력 향상  
  2. **덤벨 데드리프트** (2세트, 10회) → 허리-하체 근력 강화  
  3. **플랭크** (2세트, 30초)  
- **유산소**: 20분 빠른 걷기  

#### **토요일: 유산소 + 리커버리**  
- **운동**: 40분 가벼운 걷기 (3km/h) + 10분 요가  
- **스트레칭**: 10분 전신 스트레칭  

#### **일요일: 휴식**  
- **활동**: 가벼운 산책 (15분) + 5분 스트레칭

### [이번 주 식단 계획]  

#### **아침 (3가지 옵션)**  
1. **저칼로리**:  
   - 계란 스크램블 (2개) + 현미밥 1/2공기 + 김치  
   - **간식**: 생딸기 1개  
2. **균형형**:  
   - 두부김치찌개 (두부 100g) + 밥 1공기 + 김  
   - **간식**: 생우유 1잔 (200ml)  
3. **고칼로리**:  
   - 연어 구이 (100g) + 고구마 1/2개 + 시금치  
   - **간식**: 아몬드 10알  

#### **점심 (3가지 옵션)**  
1. **저칼로리**:  
   - 시금치나물비빔밥 (밥 1공기) + 김치  
   - **간식**: 오트밀 30g + 석류즙 100ml  
2. **균형형**:  
   - 닭가슴살 볶음밥 (닭가슴살 150g) + 김치  
   - **간식**: 흰우유 200ml + 바나나 1개  
3. **고칼로리**:  
   - 연어 스테이크 (150g) + 브로콜리 + 현미밥 1공기  
   - **간식**: 채식 프로틴바 1개  

#### **저녁 (3가지 옵션)**  
1. **저칼로리**:  
   - 채식김밥 (김 1장, 시금치, 당근, 계란 1개) + 김치  
   - **간식**: 생딸기 1개  
2. **균형형**:  
   - 닭가슴살 채소 볶음 (닭가슴살 100g, 당근, 파프리카) + 밥 1/2공기  
   - **간식**: 흰우유 200ml  
3. **고칼로리**:  
   - 연어 구이 (150g) + 고구마 1개 + 김치  
   - **간식**: 아몬드 10알

### [실천 가이드]  

#### **식사 타이밍**  
- **아침**: 7:30~8:30  
- **점심**: 12:30~13:30  
- **저녁**: 18:30~19:30  
- **간식**: 10:30, 15:30  

#### **장보기 리스트**  
- **고기**: 닭가슴살, 연어  
- **채소**: 김치, 시금치, 당근, 파프리카, 고구마  
- **곡물**: 현미, 오트밀, 김  
- **간식**: 아몬드, 프로틴바, 생딸기  

#### **Meal Prep 팁**  
- **주말에 준비**: 닭가슴살 500g을 끓여 냉동 보관, 현미 3공기 지어 냉동  
- **야채**: 김치, 시금치, 당근은 세척 후 용기 보관  
- **간편 요리**: 연어 구이, 닭가슴살 볶음밥 미리 조리 후 냉장
